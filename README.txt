vsdirent: A minimal C++ <dirent.h> implementation for Visual Studio
Written by Aggelos Kolaitis
Published under the zlib license, see LICENSE.txt

To use vsdirent, just include vsdirent.h and vsdirent.cpp in your project.
There are also Unicode variations (wchar_t) of the dirent API functions.
Requirements: Visual Studio (obviously)


Source code:
* vsdirent.h: Main header
* vsdirent.cpp: Implementation

* test.cpp: Simple test program
* testw.cpp: Simple test program, using Unicode functions
